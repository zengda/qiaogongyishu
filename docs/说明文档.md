## 一、项目规划

### 1.1 项目概述

「巧工艺墅」是一套面向农村自建房/乡村别墅设计行业的全栈解决方案，包含：

| 子系统 | 技术栈 | 用途 |
|--------|--------|------|
| 后端 API | Flask 3.0 + SQLAlchemy + Redis (port 5001) | 业务逻辑、数据存储 |
| 管理后台 | Vue 3 + Element Plus + Vite (port 3000) | 管理员管理产品/客户/分类等 |
| 微信小程序 | 原生小程序框架 | C 端用户浏览产品、提交咨询 |

### 1.2 核心目标

1. **展示设计实力** — 精美图纸产品展示，让客户直观感受设计水平
2. **客户获客转化** — 通过"领取图纸"表单收集意向客户信息
3. **客服引导成交** — 在线客服入口引导客户添加微信，深度沟通
4. **高效后台管理** — 管理后台支持产品 CRUD、客户管理、数据导出

## 二、实施方案

### 2.1 技术架构

```
客户端层
├── 微信小程序（原生 WXML/WXSS/JS）    → C 端用户浏览产品、提交表单
└── 管理后台（Vue 3 + Element Plus）   → /admin 管理员操作

网关层
└── Nginx（反向代理 + HTTPS + 静态资源）

应用层
└── Flask 3.0 (gunicorn, port 5001)
    ├── 蓝图路由（9 个蓝图模块）
    ├── JWT 认证（admin_required 装饰器）
    ├── 存储抽象层（本地存储 / 阿里云 OSS）
    └── Redis 缓存（分类/标签/Banner）

数据层
├── SQLite（开发环境，instance/qiaogongyishu.db）
└── Redis（缓存，localhost:6379）
```

### 2.2 项目目录结构

```
qiaogongyishu/
├── backend/                  # Flask 后端
│   ├── app/
│   │   ├── api/              # API 蓝图（9 个文件）
│   │   ├── models/           # ORM 数据模型（10 个文件）
│   │   ├── services/         # 业务逻辑 + 存储抽象层
│   │   └── utils/            # 工具（装饰器/分页/响应）
│   ├── scripts/              # 辅助脚本（miniprogram-ci.js）
│   ├── uploads/              # 本地文件存储
│   ├── run.py                # 启动入口
│   └── .env                  # 环境变量
├── admin/                    # Vue 3 管理后台
│   └── src/
│       ├── views/            # 页面组件（15+ 文件）
│       ├── api/index.js      # API 封装
│       ├── router/           # 路由 + 守卫
│       └── store/            # Vuex 状态管理
├── miniprogram/              # 微信小程序
│   ├── pages/                # 5 个页面
│   ├── components/           # 3 个组件
│   └── utils/                # 请求封装
└── docs/                     # 项目文档
```

### 2.3 生产环境部署方案

| 配置项 | 值 |
|--------|-----|
| 生产域名 | `qgys.rongyun.online` |
| WSGI 服务器 | Gunicorn（4 workers, timeout 120s） |
| 反向代理 | Nginx（HTTPS + 静态资源 + API 反代） |
| 启动命令 | `gunicorn -w 4 -b 127.0.0.1:5001 "app:create_app('production')"` |
| Python 版本 | 3.10.x |
| 关键环境变量 | `FLASK_ENV=production`, `SECRET_KEY`, `JWT_SECRET_KEY` |

### 2.4 开发环境启动

```bash
# 后端
cd backend && python3 run.py          # → http://localhost:5001

# 管理后台
cd admin && npm run dev                # → http://localhost:3000
```
